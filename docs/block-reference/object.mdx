## Overview

The Object Block allows you to create an object from input values and a JSON template. The block automatically escapes the input values and inserts them into the template.

This block is particularly useful for creating objects from multiple inputs. It supports any data type as input and outputs an object.

## Example 1: Creating an object from multiple inputs

1. Create an Object Block and set the JSON Template to the following:

   ```json
   {
     "name": "{{name}}",
     "age": {{age}},
     "job": "{{job}}"
   }
   ```

2. Create two Text Blocks and set their values to `John Doe` and `Engineer` respectively. Connect them to the `name` and `job` inputs of the Object Block.
3. Create a Number Block and set its value to `30`. Connect it to the `age` input of the Object Block.
4. Run the flow. The `Output` of the Object Block should be the following object:

   ```json
   {
     "name": "John Doe",
     "age": 30,
     "job": "Engineer"
   }
   ```

## Error Handling

The Object Block will error if the JSON Template is not a valid JSON string or if the interpolated JSON string (after inserting the input values) is not a valid JSON string.

## FAQ

**Q: Can I use the Object Block to create an array?**

A: Yes, you can use the Object Block to create an array by setting the JSON Template to a valid JSON array string. For example:

```json
["{{value1}}", "{{value2}}", "{{value3}}"]
```

**Q: How are the input values escaped?**

A: The input values are automatically escaped by the Object Block. If the input value is a string, it is escaped using `JSON.stringify()`. If the input value is not a string, it is first converted to a string using `JSON.stringify()` and then escaped.
