## Overview

The Pop Block is used to remove either the first or last element from an array and output both the removed element and the remaining array. By default, the Pop Block will pop the last element from the array.

This block is useful when you want to process elements of an array one by one in a loop.

## Example 1: Pop the last element from an array

1. Create an [Array Block](./array.mdx) and set the values to `["John", "Doe", "30", "Engineer"]`.
2. Create a Pop Block and connect the Array Block to its `Array` input.
3. Run the flow. The `Last` output of the Pop Block should be `"Engineer"` and the `Rest` output should be `["John", "Doe", "30"]`.

![Pop Block Example 1](./assets/pop-block-example-01.png)

## Example 2: Pop the first element from an array

1. Create an [Array Block](./array.mdx) and set the values to `["John", "Doe", "30", "Engineer"]`.
2. Create a Pop Block, enable the `Pop from front` setting, and connect the Array Block to its `Array` input.
3. Run the flow. The `Last` output of the Pop Block should be `"John"` and the `Rest` output should be `["Doe", "30", "Engineer"]`.

![Pop Block Example 2](./assets/pop-block-example-02.png)

## Error Handling

The Pop Block will error if the `Array` input is not provided, is not an array, or is an empty array.

## FAQ

**Q: Can I pop multiple elements from an array at once?**

A: No, the Pop Block can only pop one element at a time. If you want to remove multiple elements from an array, you can use multiple Pop Blocks in a sequence.

**Q: Can I use the Pop Block to pop elements from a string?**

A: No, the Pop Block only works with arrays. If you want to remove characters from a string, you can use a [Code Block](./code.mdx) with custom JavaScript code.

**Q: What if the array is empty?**

A: The Pop Block will error if the array is empty. You can use an [If Block](./if.mdx) to check if the array is empty before popping from it.
