# All Blocks

The following is a comprehensive list of all blocks built into the core of Lawme.

## Text

<CardGroup cols={2}>
  <Card title="Chunk" icon="puzzle-piece" href="./chunk.mdx">
    Splits a string into an array of strings based on a token count.
  </Card>
  <Card title="Extract with Regex" icon="magnifying-glass" href="./extract-with-regex.mdx">
    Extracts strings from a string using a regular expression.
  </Card>
  <Card title="Join" icon="link" href="./join.mdx">
    Concatenates an array of strings into a single string.
  </Card>
  <Card title="Prompt" icon="comment" href="./prompt.mdx">
    Creates a chat message with an attached "type" and optional "name".
  </Card>
  <Card title="Split Text" icon="scissors" href="./split-text.mdx">
    Splits a string into an array of substrings based on a delimiter.
  </Card>
  <Card title="Text" icon="font" href="./text.mdx">
    Outputs a string of text with support for interpolation.
  </Card>
</CardGroup>

## AI

<CardGroup cols={2}>
  <Card title="Chat" icon="robot" href="./chat.mdx">

  </Card>
</CardGroup>

## Lists

<CardGroup cols={2}>
  <Card title="Array" icon="layer-group" href="./array.mdx">
    Constructs or merges arrays from multiple inputs.
  </Card>
  <Card title="Filter" icon="filter" href="./filter.mdx">
    Filters elements of an array based on boolean values.
  </Card>
  <Card title="Pop" icon="arrow-up-from-bracket" href="./pop.mdx">
    Removes the first or last element from an array.
  </Card>
  <Card title="Shuffle" icon="shuffle" href="./shuffle.mdx">
    Randomizes the order of elements in an array.
  </Card>
  <Card title="Slice" icon="knife" href="./slice.mdx">
    Extracts a specific portion of an array.
  </Card>
</CardGroup>

## Numbers

<CardGroup cols={2}>

  </Card>
  <Card title="Number" icon="hashtag" href="./number.mdx">

  <Card title="RNG" icon="dice" href="./rng.mdx">

  </Card>

## Logic

<CardGroup cols={2}>
  <Card title="Abort Flow" icon="stop" href="./abort-flow.mdx">
    Immediately stops the execution of the entire flow.
  </Card>
  <Card title="Coalesce" icon="code-merge" href="./coalesce.mdx">
    Returns the first non-null value from a list of inputs.
  </Card>
  <Card title="Compare" icon="not-equal" href="./compare.mdx">
    Performs comparison operations between two input values.
  </Card>
  <Card title="Delay" icon="clock" href="./delay.mdx">
    Introduces a pause in the execution of the flow.
  </Card>
  <Card title="If" icon="code-branch" href="./if.mdx">
    Passes a value through True or False port based on a condition.
  </Card>
  <Card title="If/Else" icon="code-fork" href="./if-else.mdx">
    Chooses between two values based on a condition.
  </Card>
  <Card title="Match" icon="equals" href="./match.mdx">
    Matches a string input against regular expressions.
  </Card>
  <Card title="Passthrough" icon="right-left" href="./passthrough.mdx">
    Transfers the input value directly to the output.
  </Card>
  <Card title="Race Inputs" icon="flag-checkered" href="./race-inputs.mdx">
    Outputs the value of the first input to complete.
  </Card>
</CardGroup>
